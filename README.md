# PrintGraph - Приложение для визуализации данных и печати графиков

В ходе лабораторной работы необходимо:

- Реализовать загрузку данных из SQLite и JSON;

- Построить графики по загруженным данным;

- Обеспечить возможность выбора различных типов графиков;

- Реализовать переключение визуального оформления графиков (цветной/черно-белый);

- Реализовать экспорт графиков в PDF;

- Использовать архитектуру MVC;

- Использовать внедрение зависимостей через IoC-контейнер.

## Предлагаемая архитектура
### Концепция программирования MVC
MVC включает в себя 3 типа объектов:
• Модель – осуществляет соединение с источником данных, предоставляет интерфейс другим компонентам архитектуры.
• Вид (представление) – обеспечивает конечное представление данных для пользователя.
• Контроллер - отвечает за пользовательский интерфейс.

|Компонент|	Реализация|	Назначение|
| --- | --- | --- |
|Model	|DataPoint, GraphModel	|Хранение и управление данными|
|View	|ThemeWidget, QTreeView, QTableView	|Отображение данных пользователю|
|Controller	|MainWindow	|Обработка действий пользователя, связь Model и View|

### Паттерн Адаптер (Adapter)
Паттерн Адаптер позволяет объектам с несовместимыми интерфейсами работать вместе. При этом адаптер оборачивает один из объектов, так что
другой объект даже не знает о наличии первого.

## Функционал программы
Приложение предоставляет пользователю следующие возможности:

### Работа с файлами
- Просмотр файловой системы компьютера;

- Выбор каталога через кнопку "Выбрать папку";

- Отображение содержимого выбранной папки в таблице;

- Выбор SQLite-файла или JSON-файла.

### Работа с источниками данных

- Определение таблицы данных и её колонок;

- Чтение данных;

- Преобразование данных в структуру DataPoint (дата + значение).

### Построение графиков
- Отображение линейного графика (LineChart);

- Отображение столбчатой диаграммы (BarChart);

- Автоматическое обновление графика после выбора файла.

### Изменение цвета графика
- Цветной режим (по умолчанию);

- Черно-белый режим (переключается чекбоксом).

### Сохранение графика
- Экспорт графика в PDF;

- Выбор пользователем пути сохранения файла
